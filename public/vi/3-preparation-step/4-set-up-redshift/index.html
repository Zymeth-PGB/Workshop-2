Thiết lập và kết nối tới Redshift

Tạo Redshift Cluster

  1. Nhập và tìm kiếm Redshift trong thanh tìm kiếm sau đó ấn chọn Redshift.

  2. Ở thanh điều hướng bên trái chọn Cluster.

  3. Chọn Create Cluster để tạo một Cluster mới.

  4. Nhập tên Cluster và chọn các tùy chọn như hình, do mục tiêu là làm để tìm hiểu nên ta chọn ra3.xlplus để tiết kiệm chi phí.

  5. Nhập tên cho Admin User và chọn nhập mật khẩu thủ công để đặt mật khẩu của bạn.

    Username và password này được sử dụng để tạo kết nối tới Query Editor nên bạn cần ghi nhớ để có thể kết nối đến Query Editor.

  1. Sử dụng các cài đặt bổ sung mặc định và ấn Create Cluster để tạo Cluster.

  2. Sau khi tạo xong Cluster, nhìn vào thanh điều hướng bên trái và chọn Query Editor V2.

  3. Tại màn hình Query Editor chọn vào dấu 3 chấm sau đó chọn Create connection để tạo kết nối tới Query Editor.

  4. Khi chọn Create connection sẽ hiển thị lên màn hình các lựa chọn để liên kết đến Query Editor, ta chọn như hình và điền Username, password đã tạo trước đó vào.

  5. Sau khi tạo kết nối đến Query Editor thành công ta sẽ dán Script SQL được tạo sẵn ở Github và ấn run để tạo ra table có các trường dữ liệu phù hơp với dữ liệu sẽ được load vào.

  6. Tạo thành công table thì ta sẽ quay trở lại thanh điều khiển bên trái của Glue và chọn connection.

  7. Chọn Create connection để tạo connection tới Redshift.

  8. Ở bước đầu tiên tìm kiếm data source là Redshift sau đó ấn chọn và ấn next.

  9. Chọn Database instances là cluster vừa tạo và nhập Username password, sau đó chọn next.

  10. Đặt tên cho connection và chọn next.

  11. Kiểm tra lại các cài đặt đã đúng chưa và chọn Creation connection.

  12. Tạo xong sẽ có connection tới Redshift như hình.
